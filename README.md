# Resume App

Currículo profissional interativo desenvolvido como aplicação web com **Streamlit** e **Python**.

---

## Sobre o projeto

Uma alternativa ao currículo tradicional em PDF — apresentado como uma aplicação interativa com abas organizadas por seção, barras de progresso para visualizar habilidades técnicas e expansores para detalhar experiências profissionais.

---

## Funcionalidades

- Navegação por abas: Sobre, Habilidades, Experiência, Educação, Projetos e Contato
- Visualização de habilidades técnicas com barras de progresso
- Experiências profissionais em expansores organizados por período
- Formação acadêmica e projetos em seções dedicadas
- Links diretos para LinkedIn e GitHub

---

## Tecnologias

| Tecnologia | Uso |
|---|---|
| Python | Linguagem principal |
| Streamlit | Framework de interface web |
| Pandas | Estruturação dos dados de habilidades |
